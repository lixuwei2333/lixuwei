你的名字题解

Veröffentlicht am 2017-11-01

老师需要我写篇题解
题号3700
题意：
给定一个日期和对应的星期，求3年为星期几。

数据范围：
输入数据有多组(数据组数不超过 50000)，保证输入日期不含 2 月 29 日的情况，且其年份在 1900 到 2035 之间。

变量说明：
year,month,day,week 分别对应输入的年月日星期。

分析：
1.连续4年之中最多有1年为闰年
2.从该日期到3年前跳跃了1095(365+365+365，没有经过2月29日)或1096(365*2+366，经过2月29日)天
3.星期7天一循环，1095%7=3，1096%7=4，所以结果为week-3或week-4
4.year-1或year-2为闰年时，恒经过2月29日，既week-4
5.当year或year-3为闰年时，按month分情况讨论

特殊数据：
当year为1903,1902等年份时，前3年加本年无闰年，此时经过1095天。

using namespace std;
int ly(int year) //判断闰年
{
if((year%400==0)||(year%4==0&&year%100!=0))
return 1;
else return 0;
}
int year,month,day,week,flag;
int main()
{
while(cin>>year>>month>>day>>week) //读入数据
{
flag = week; //记录当前星期，为了处理4年无闰年的情况
if (ly(year)) //当year为闰年时
{
if (month >= 3) week -= 4; //分类讨论
if (month <= 2) week -= 3;
}
if (ly(year - 3)) //当year-3为闰年时
{
if (month >= 3) week -= 3; //注意这里和上面不一样，自行思考
if (month <= 2) week -= 4;
}
if (ly(year-1)||ly(year-2)) //当year-1或year-2为闰年时
{
week -= 4;
}
if(week == flag) week -= 3; //等价于当4年不存在闰年时(上面所有if条件都没触发)
if (week <= 0) week += 7; //防止week出现负数的必要处理
cout<<week<<endl; //输出week加换行符
}
return 0;
}
